# Terraform meta-arguments

- Terraform provides below meta-arguments that we can use.
  * **Depends_on:** 
       * Use the depends_on meta-argument to handle hidden resource or module dependencies that Terraform cannot automatically infer. 
  * **Count:** 
       * Use the count meta-argument to create or manage several similar obejects.
  * **For_each:**
       * Use the for_each meta-argument to create or manage several similar obejects.
  * **Provider:**
       * Use the provider meta-argument to override terraform's default provider configuration.
  * **Lifecycle:**
       * The lifecycle meta-argument defines the lifecycle for the resource.
